# Что такое язык программирования?

**Язык программирования** — это формальный язык, предназначенный для написания компьютерных программ. По сути, это набор правил и символов, с помощью которых человек может передать компьютеру четкие инструкции (алгоритмы).

## Как это работает?
Компьютеры понимают только бинарный код (нули и единицы). Писать на нем напрямую крайне сложно, поэтому были созданы языки программирования:
1. **Исходный код:** Программист пишет текст на понятном языке (например, Python или C++).
2. **Транслятор (Компилятор/Интерпретатор):** Специальная программа переводит этот текст в машинный код.
3. **Исполнение:** Процессор выполняет полученные команды.

## Основные виды языков
*   **Низкоуровневые (Assembler):** Близки к машинному коду, дают полный контроль над «железом», но сложны в освоении.
*   **Высокоуровневые (Python, JS, Java):** Похожи на человеческий английский язык. Они проще в написании и чтении.

## Зачем их так много?
Каждый язык заточен под свои задачи:
- **JavaScript** — для интерактивности сайтов.
- **Python** — для анализа данных и ИИ.
- **C++** — для высокопроизводительных игр и систем.
- **Swift/Kotlin** — для мобильных приложений.

*Программирование — это не просто написание кода, это искусство решения задач.*
