# What is a Programming Language?

A **programming language** is a formal language designed to write computer programs. Essentially, it is a set of rules and symbols that allow a human to provide clear instructions (algorithms) to a computer.

## How does it work?
Computers only understand binary code (zeros and ones). Writing in binary is extremely difficult, so programming languages were created:
1. **Source Code:** A programmer writes text in a human-readable language (e.g., Python or C++).
2. **Translator (Compiler/Interpreter):** A special program converts this text into machine code.
3. **Execution:** The processor executes the received commands.

## Main Types of Languages
*   **Low-level (Assembly):** Close to machine code, providing full control over hardware but difficult to learn.
*   **High-level (Python, JS, Java):** Similar to human English. They are much easier to read and write.

## Why are there so many?
Each language is tailored for specific tasks:
- **JavaScript** — for website interactivity.
- **Python** — for data analysis and AI.
- **C++** — for high-performance games and systems.
- **Swift/Kotlin** — for mobile applications.
